Pesquisa e Resumo Visual: Formulários em HTML e CSS

1. Conceitos de Formulários

- O que são formulários em HTML? São estruturas usadas para coletar dados do usuário em uma página web. Eles são definidos pela tag < form> e podem conter campos como caixas de texto, botões, seleções e outros elementos de interação.

- Para que servem? Servem para coletar dados e interações dos usuários em um site e enviá-los para um servidor para processamento. Eles funcionam como uma ponte entre o usuário e o sistema, sendo usados em situações como login, cadastro, envio de mensagens, buscas, upload de arquivos e pesquisas.

Exemplo de formulário de login em uma página web, onde o usuário insere seus dados para acessar o sistema.

2. Tag < form>

- Função: Essa tag é responsável por definir um formulário em HTML. Ela envolve todos os campos e permite o envio dos dados inseridos pelo usuário.

Atributos principais:

- action: Define para onde são enviados os dados do formulário quando um formulário é enviado.

- method (GET e POST): O atributo method recebe como valor o método HTTP que o formulário irá utilizar para enviar os dados. HTTP (HyperText Transfer Protocol) é o protocolo responsável pela comunicação entre o navegador e o servidor. Entre os métodos mais utilizados estão o GET e o POST: o GET envia os dados pela URL, sendo menos seguro e mais utilizado em buscas, enquanto o POST envia os dados de forma mais segura, sem exibi-los na URL, sendo utilizado em situações como login e cadastro.

Como funciona o envio de dados? O envio de dados em um formulário HTML ocorre quando o usuário preenche os campos (como < input>, < textarea> e < select>) e clica no botão de envio. Nesse momento, as informações são coletadas e enviadas para um servidor, arquivo ou script definido no atributo action. Esse processo é controlado pelos atributos da tag < form>, que determinam para onde os dados serão enviados e como isso será feito (GET ou POST).

Exemplo de código de um formulário HTML, onde a tag < form> define o envio de dados (com action e method="POST"), enquanto os campos < input> e < label> permitem a entrada de informações do usuário, que são enviadas ao servidor ao clicar no botão de envio.

3. Principais Elementos de Formulário

< input> - Cria campos de entrada em formulários para que o usuário ou usuária possa inserir algum texto, data, número, cor, etc. O atributo type define o tipo de informação que será inserida, podendo assumir valores como text para textos simples, password para ocultar caracteres, email para validação de e-mails, number para números, checkbox para seleção de múltiplas opções, radio para escolha única, file para envio de arquivos e submit para envio do formulário. Dessa forma, essa tag é essencial para a construção de formulários interativos na web.

Exemplos:

< input type="text">

< input type="password">

< input type="email"

< input type="number">

< label> - É utilizada para definir o "rótulo" do input, indicando ao usuário qual informação deve ser inserida.

Exemplo:

< label for="nome"> Insira seu nome: < /label>

< input type="text" id="nome">

< textarea> - Representa uma caixa de texto, utilizada quando se deseja permitir que o usuário insira textos mais longos, como um comentário ou formulário de retorno. Diferente do < input>, ela permite múltiplas linhas de texto, sendo muito usada em formulários de contato ou feedback.

Exemplo:

< textarea placeholder="Digite sua mensagem"> < /textarea>

*O atributo placeholder especifica uma breve dica que descreve o valor esperado de uma área de texto.

< select> e < option> - São utilizadas para criar listas de opções em formulários. A tag < select> define o campo de seleção, enquanto a tag < option> define cada uma das opções disponíveis para o usuário escolher.

Exemplo:

< select id="cidade">

< option> Joinville < /option>

< option> Curitiba < /option>

< /select>

< button> - Representa um botão clicável em um formulário, e possui como principal atributo o type, que define o comportamento do botão. Quando o valor type="submit" é utilizado dentro de um formulário, o botão envia os dados preenchidos pelo usuário ao servidor.

Exemplo:

< button type="submit"> Enviar < /button>
